Esta aplicación web representa un control de acceso y gestión de usuarios básico. Está desarrollada bajo el framework Flask (Python).

El único equisito previo es tener Docker instalado en el sistema.

Para ejecutarla hay que construir la imagen de Docker abriendo la terminal en la ubicación del DOckerfile e introduciendo la siguiente orden: 'docker compose up --build -d'. Y para lanzarla esta orden: 'docker build -d'.

Se accederá a la aplicación desde el navegador en la  siguiente dirrección web: 'http://localhost:5001'.
